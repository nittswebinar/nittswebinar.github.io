Plugin WordPress Get Email

Un plugin WordPress pour les utilisateurs SG-Autorépondeur. Commencez à capturer les adresses email sur votre blog WordPress en moins de 2 minutes. Le plugin facilite la mise en place d'un formulaire de capture email, qui enregistre automatiquement les nouveaux contacts dans votre compte SG-Autorépondeur. Le but : Vous éviter de vous arracher les cheveux pour la technique.
